# Dia a Dia - Python Practice

> Repositório de estudos e prática diária de Python para manter minhas habilidades de desenvolvimento afiadas

## Sobre o Projeto

Este é meu espaço pessoal de aprendizado contínuo em Python, onde pratico **lógica de programação** e conceitos fundamentais que uso no **dia a dia como desenvolvedor**.

### Por que este projeto existe?

- **Manter a prática afiada**: Como desenvolvedor, é fundamental exercitar constantemente a lógica e os fundamentos
- **Aprender conceitos básicos**: Cada exercício é uma oportunidade de dominar melhor recursos nativos do Python
- **Consolidar conhecimentos**: Através da repetição e variação, solidificar o que já sei e descobrir novas abordagens
- **Registro de evolução**: Documentar minha jornada e crescimento técnico ao longo do tempo

Os exercícios são voltados para problemas práticos e situações reais do dia a dia de desenvolvimento, como manipulação de dados, validações, transformações e otimizações.

### Filosofia

O objetivo não é fazer um exercício por dia, mas sim **praticar de forma consistente**. Alguns dias faço um exercício, outros dias faço vários - o importante é manter a constância, registrar meu progresso e sempre buscar melhorar minhas soluções.

## Estrutura do Projeto

Cada arquivo `.py` contém:
- Contexto do problema
- Objetivo do exercício
- Entrada esperada
- Saída esperada
- Solução implementada
